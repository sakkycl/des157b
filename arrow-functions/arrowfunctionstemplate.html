Popup event function

Every time a user clicks on the Leaflet map, the coordinates will be revealed in a popup. I got this function from my Libaries Challenge in the Toolkit

Named Function

            function onMapClick(e) {
                popup
                    .setLatLng(e.latlng)
                    .setContent("You clicked the map at " + e.latlng.toString())
                    .openOn(map);
            }

Function Expression

            const popupEvent = function(onMapClick(e)){
                popup
                    .setLatLng(e.latlng)
                    .setContent("You clicked the map at " + e.latlng.toString())
                    .openOn(map);
            }

Arrow Function

            const popupEvent = (onMapClick(e)) => {
                popup
                    .setLatLng(e.latlng)
                    .setContent("You clicked the map at " + e.latlng.toString())
                    .openOn(map);
            }
